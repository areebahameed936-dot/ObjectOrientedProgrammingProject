# ObjectOrientedProgrammingProject
The Library Management System is a desktop-based application developed in Java using Swing and AWT packages. The project is designed to automate and simplify the management of library operations such as adding, updating, deleting, and managing accounts and records through a graphical user interface (GUI). 
The project demonstrates important Object-Oriented Programming (OOP) concepts including:

Inheritance
Abstraction
Encapsulation
Polymorphism
Composition
Event Handling

The application uses various GUI components such as:

JFrame
JPanel
JButton
JLabel
JTextField
JTextArea

These components help create an interactive and visually appealing interface. Buttons like Add, Update, Delete, and Back allow users to perform different operations easily.

The system improves:

Record management efficiency
Data organization
User interaction
GUI development experience
Code maintainability and modularity
